HF

Heart Failure Prediction Dataset

A clean summary of the dataset used for heart disease / heart failure prediction tasks.

About Dataset

Cardiovascular diseases (CVDs) are the number 1 cause of death globally — an estimated 17.9 million lives each year (~31% of all deaths). Heart failure often results from CVDs. This dataset contains 11 features that can be used to predict possible heart disease and is a combined, cleaned version of multiple heart datasets for research purposes.

Context

People with cardiovascular disease or at high cardiovascular risk (hypertension, diabetes, hyperlipidaemia, etc.) need early detection and management. Machine learning models can assist clinicians by flagging high-risk patients for further evaluation.

Attribute Information

AgeAge of the patient (years)
SexM: Male, F: Female
ChestPainTypeTA, ATA, NAP, ASY
RestingBPResting blood pressure (mm Hg)
CholesterolSerum cholesterol (mg/dl)
FastingBS1 if >120 mg/dl, else 0
RestingECGNormal, ST, LVH
MaxHRMaximum heart rate achieved (60–202)
ExerciseAnginaY / N
OldpeakST depression measured (numeric)
ST_SlopeUp / Flat / Down

Target

HeartDisease — output class: 1 means heart disease; 0 means normal.

Source & Notes

This dataset was created by combining five heart datasets (Cleveland, Hungarian, Switzerland, Long Beach VA, Stalog) and then cleaning duplicates. Final dataset: 918 observations. For the original index of heart disease datasets, see the UCI repository.
